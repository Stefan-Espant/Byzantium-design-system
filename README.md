# Byzantium

> A structured, accessible design system built on semantic tokens,
> reusable components and clear patterns.
> The single source of truth for everything visual.

## Packages

| Package | Description |
|---------|-------------|
| `@byzantium/core` | Tokens, components, composables |

## Structure

Byzantium is built in three layers:

- **Primitives** — ByzButton, ByzInput, ByzIcon
- **Composites** — ByzAlert, ByzCard, ByzModal
- **Patterns**   — ByzHero, ByzNavbar

## Getting started

```bash
pnpm install
pnpm dev
```

## Used by

- [Vienna](https://github.com/you/vienna) — Nuxt 4 boilerplate
