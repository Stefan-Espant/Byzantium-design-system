Add ByzRating, ByzCode, ByzKbd primitives; /forms and /typography playground pages

- New `ByzRating` primitive: star rating with v-model, readonly mode, sm/md/lg sizes, ARIA radiogroup
- New `ByzCode` primitive: monospace code block with language badge and copy-to-clipboard button
- New `ByzKbd` primitive: keyboard key display with multi-key support and physical key styling
- New `/forms` playground page: all form components + live `useForm` demo with validation and toast on submit
- New `/typography` playground page: font specimens (sans/serif/mono), type scale, weights, leadings, text hierarchy
- Navigation updated in all 7 playground pages (desktop + mobile)
- Locale strings added for all 16 languages (`navForms`, `navTypography`, `formsPageTitle`, `typographyPageTitle`)
